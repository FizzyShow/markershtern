## Context: 

Микросервис отвечающий за портфель пользователя(взаимодействие с БД через API микросервиса)

## Decision

БД микросервиса будет хранить информацию о портфеле пользователя. Портфель пользователя(в виде таблицы) будет выглядеть следующим образом:(

id пользователя,
название продукта,
кол-во продукта)


Микросервис будет содержать следующий API:
Post method:
1.Добавление продукта в портфель, запрос который принимает следующие данные:(id пользователя, название продукта, кол-во продукта).\

Get method(тип возвращаемых данных JSON):
1.Получить определенный продукт, запрос принимает следующие данные:(id пользователя, название продукта).

2.Получить список продуктов(всего портфеля), запрос принимает следующие данные:(id пользователя).
